Lesson 1: Introduction to HTML

What is HTML?

HTML stands for Hypertext Markup Language. It's the standard markup language used to create web pages. HTML provides the structure and layout of a web page by using a variety of tags and attributes to define elements such as headings, paragraphs, images, links, and more. It works in conjunction with cascading style sheets (CSS) and JavaScript to build interactive and visually appealing web pages.

HTML Document layout

• The declaration specifies that the document is an HTML5 document.
• The element is the root element.
• The section contains metadata, including the document's character set, title, link to an external stylesheet (styles.css), and a script (script.js).
• The section contains visible content such as headings (h1), paragraphs (p), images (img), and a hyperlink (a).

HTML Tags

HTML Tags have a starting tag - content - ending tag

Starting tag: <h1>
Ending tag: </h1>


HTML Headers

HTML <h> elements are used to define headings in a webpage. These elements range from <h1> to <h6>, with <h1> being the highest level (most important) heading and <h6> being the lowest level (least important) heading.

• <h1>: This is the main heading of a page or a section. It's usually the largest and most prominent heading on the page. Use it sparingly to represent the main topic or title of your content.

• <h2>: These are used for subheadings within sections created by <<h1> headings. They are slightly smaller than <h1> headings and can be used to divide content into smaller chunks.

• <h3> to <h6>: These are progressively smaller headings used for sub-subheadings and so on. They are used to further organize and structure content hierarchically.
